Base_data of students
Simple base_data students system in Python using a list of dictionaries: {"name": str, "id": int, "age": int, "program": str, "state": str }

Main Features
Add students
Show students
Search students
Update students
Delete students
app.py: Main menu and program loop.
menu.py: Menu text.
validation.py: Input validations.
Add_student.py: Add student function.
show_student.py: Show student function.
search_student.py: Search student function.
update_student.py: Update student function.
delete_student.py: Delete student function.

Run

Run from the project folder: app.py
